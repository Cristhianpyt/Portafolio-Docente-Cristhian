## 🔬 Modelo LEPI — Aporte Académico Original

```
L → Liderazgo     Distribuido, transformacional, medible con datos
E → Educativo     Centrado en el aprendizaje real y contextual
P → Praxeológico  Ver → Juzgar → Actuar → Devolver (Marco Juliao, UNIMINUTO)
I → Innovador     Tecnología + IA + datos al servicio del cambio pedagógico
```

### Los 4 Momentos Praxeológicos (Juliao, 2017)
| Momento | Pregunta | Acción del Docente |
|---|---|---|
| **VER** | ¿Qué pasa? | Observar, diagnosticar, sistematizar |
| **JUZGAR** | ¿Por qué pasa? | Analizar críticamente, reflexionar |
| **ACTUAR** | ¿Qué hacemos? | Implementar estrategias innovadoras |
| **DEVOLVER** | ¿Qué aprendimos? | Retroalimentar, evaluar el impacto |
